# AgExcel

AgExcel is an Angular Wrapper for [JExcel V3](https://www.npmjs.com/package/jexcel).

## Installation
###### Install ag-excel
`npm i ag-excel`
###### Import Assets
Import the assets in angular.json
```json
{
  "architect": {
    "assets": [
		{
		  "glob": "**/*",
		  "input": "./node_modules/ag-excel/assets/AgExcel",
		  "output": "./assets/AgExcel"
		}
	]
  }
}
```
